# **App Name**: SnipQueue

## Core Features:

- Appointment Booking: Allow customers to book appointments for specific services and time slots.
- Service Selection: Display a list of available services with descriptions and prices.
- Barber Availability: Show available barbers and their schedules for appointment booking.
- Queue management tool: AI-powered LLM will intelligently manage the queue by rescheduling appointments to optimize barber utilization.
- Appointment Confirmation: Send automated email or SMS confirmations and reminders to customers.
- Admin Dashboard: Provide an admin interface for managing barbers, services, schedules, and appointments.
- Customer Profiles: Allow customers to create and manage their profiles with contact information and appointment history.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a professional and trustworthy feel.
- Background color: Light gray (#F5F5F5) for a clean and modern look.
- Accent color: Teal (#009688) for interactive elements and highlights.
- Body and headline font: 'PT Sans', a humanist sans-serif.
- Use clean, minimalist icons for services and UI elements.
- Use a grid-based layout for clear information hierarchy and easy navigation.
- Subtle animations for transitions and feedback on user interactions.
